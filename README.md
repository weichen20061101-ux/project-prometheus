# Project Prometheus

Project Prometheus is a React + Tailwind learning system that turns long-term study into daily RPG-style quests.

## Core flow

- A required daily gate quest from the `Analyst Path`
- Topic unlocks after the gate quest is completed
- Topic-based EXP and level progression
- Local progress persistence with `localStorage`
- Static deployment through GitHub Pages

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

The production build is generated in `dist/`.
